## Especificação: Blocos da Agenda

### Estrutura de um bloco (box drawing)

```
HH:MM ─┬───────────────────────────────╮   abertura (topo do bloco)
       │ Nome           Status   Dur   │   conteúdo (nome, status, duração)
HH:MM ─┤ ░░░░░░░░░░░░░░░░░░░░░░░░░░░░  │   fill bar (1 linha por 30min)
       ├───────────────────────────────╯   fechamento → próximo bloco (├)
       ╰───────────────────────────────╯   fechamento → último bloco  (╰)
```

### Conectores da régua de tempo

| Char | Uso                                         |
| ---- | ------------------------------------------- |
| ─┬─  | Início de bloco                             |
| ─┤   | Slot intermediário (fill bar)               |
| ─┼─  | Transição: fim de bloco → início do próximo |
| ─┤   | Fechamento com próximo bloco (├──╯)         |
| │    | Slot vazio ou intermediário sem borda       |
| ╰──╯ | Fechamento do último bloco                  |

### Proporcionalidade (D-001)

- 30min = 1 linha de fill
- 1h = 2 linhas de fill
- 1h30 = 3 linhas de fill
- Mínimo: 1 linha de fill (para blocos < 30min)

### Horários livres

```
│               ┈ livre ┈
```

Exibido entre blocos quando há gap >= 30min.

### Indicadores por urgência

### Colunas

```
  ind  nome                  data      horário
  !    Relatório Q1          Ontem     14:00
  !    Review PR #142      Hoje      --:--
  ·    Email cliente         Sex 21    09:00

```

- `--:--` quando não há horário definido
- Data relativa: Ontem, Hoje, Amanhã, Dia da semana + dia, ou "X sem"

### Seções (space-between)

| Posição  | Conteúdo                          |
| -------- | --------------------------------- |
| Esquerda | Nome da rotina ativa              |
| Centro   | Data abreviada + hora atual       |
| Direita  | Timer ativo (se houver) + elapsed |

- Hora atualiza a cada minuto
- Timer atualiza a cada segundo
- Sem rotina: "[Sem rotina]"
- Sem timer: seção direita vazia
